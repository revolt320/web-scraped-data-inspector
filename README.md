# Data Inspector (Streamlit)

A lightweight data inspection and profiling tool built with Streamlit, designed to quickly identify common issues in web-scraped datasets.

It helps analysts and scraping engineers explore, validate, and understand datasets before downstream processing or delivery.




## Features

- **File Upload**
  - Supports CSV, Excel, and JSON formats

- **Basic Dataset Information**
  - Filename
  - Row count and column count
  - Dataset summary
  - Column names and data types

- **Data Preview**
  - Scrollable or paginated dataset view

- **Formatting Checks**
  - Detection of extra leading/trailing spaces
  - HTML tag presence
  - Unicode and special character checks

- **Unique Value Analysis**
  - Identification of unique values per column
  - Helps detect site updates or structural data changes

- **Filtering**
  - Filter data using conditions such as:
    - Equals
    - Contains

- **Explore (Sampling)**
  - Random or rule-based sampling for quick exploration
 
- **Group & Summarize**
  - Group-by operations for summarizing data
  - Aggregate functions such as count, sum, and averages
